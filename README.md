# Notes App (Django)

A simple Django web application where users can create, edit, delete,
and search personal notes.

## Features
- User authentication (register, login, logout)
- Create, edit, delete notes
- Search notes by title or content
- Secure user-owned data
- Clean UI with Django templates and CSS

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS

## What I Learned
- Django MVC-style architecture
- Template inheritance
- Forms and validation
- Authentication and authorization
- Object-level permissions
- Query filtering and search
